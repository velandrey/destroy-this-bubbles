
- GameEngine
	- Основной класс, здесь запускается игровой цикл, здесь запускается инициализация всех игровых объектов. А также содержатся вспомогательные методы по работе с различной логикой игры.
	- Пример инициализации движка:
- ```js
			  const engine = new GameEngine(canvasRef.current);
	```
  - Создается экземпляр игрового движка с опциями указанными в объекте gameSettings
```js
			    export const gameSettings = {
    // Размеры мишени
    circle: {
        minRadius: 1, // стартовый радиус при появлении
        maxRadius: 15, // максимальный радиус (до которого растёт)
        growthSpeed: 5, // скорость увеличения радиуса (px в секунду)
        color: 'red',
    },
    // Спавн мишеней
    spawn: {
        interval: 1000, // время между появлениями (мс)
        maxCircles: 5, // максимальное количество активных кругов
    },
    // Параметры игры
    game: {
        backgroundColor: '#111', // фон канваса
        scoreOnHit: 1, // очки за попадание
        scoreOnMiss: -1, // штраф за промах
        gameDuration: 30000, //продолжительность игры в мс
    },
};
```
		
-  В конструкторе класса создаются:
	- Игровое поле canvas
	- Экземпляр логики спавна мешеней
	- Обработчик попаданий
	- Положения canvas
- Класс gameEngine содержит методы:
	- start()
		- Инициализаци игрового цикла и старт игры.
	- update()
		- Логика обновления игрового паля
	- draw()
			- Логика отрисовки объектов
	- endGame
		- Логика окончания игры
		- Сброс игры в первоначальное состояние
	- loop()
		- Игровой цикл
		- В игровом цикле вычисляется deltatime для плавности анимации
		- Обратный отсчет до завершения игры
		- Апдейт игровой логики
		- Отрисовка состояния игры
	- handleClick()
		- Для обработки кликов по игровому полю.
